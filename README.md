# Screen Time & Digital Wellness Analysis

## Project Overview
This project investigates the relationship between technology usage and mental well-being using a behavioral dataset tracking phone addiction metrics. The primary objective was to determine how daily screen time and sleep patterns correlate with addiction severity, while exploring demographic trends across different genders.

* **Language:** SQL (Structured Query Language)


##  Core Hypotheses & Analytical Goals

### 1. The Screen Time vs. Addiction Link
* **Objective:** Validate the correlation between daily phone usage and self-reported or calculated addiction levels.
* **Finding:** As hypothesized, the data reveals a direct, positive correlation: higher daily screen time hours consistently trend alongside elevated addiction severity scores.

### 2. Digital Impact on Sleep Hygiene
* **Objective:** Analyze how high screen time and addiction levels affect physical health, specifically tracking daily sleep hours.
* **Approach:** Segmented average sleep duration against varying levels of phone dependency to pinpoint the threshold where technology severely impacts sleep.

### 3. Demographic Segmentation (Gender Trends)
* **Objective:** Identify if screen time habits and vulnerability to phone addiction differ by gender.
* **Approach:** Aggregated screen time averages and addiction risk categories across demographic data to isolate any statistically relevant variances.
